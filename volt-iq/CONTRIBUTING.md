# Contribuer à Volt-IQ

Merci de l'intérêt que vous portez à Volt-IQ ! Voici comment participer.

## Par où commencer ?

1. **Lire le README** pour comprendre le projet
2. **Explorer les issues** sur GitHub — les issues taguées `good first issue` sont idéales pour débuter
3. **Forker** le dépôt et créer une branche pour votre contribution

## Types de contributions bienvenues

- 🐛 Signalement de bugs
- 💡 Suggestions de fonctionnalités
- 📖 Amélioration de la documentation
- 🔌 Ajout de nouveaux protocoles (ex: KNX, BACnet)
- 🏢 Partage de profils de sites (fichiers JSON)
- 🌍 Traductions (arabe, anglais...)

## Processus

```bash
# 1. Forker et cloner
git clone https://github.com/VOTRE_COMPTE/volt-iq.git

# 2. Créer une branche
git checkout -b feature/nom-de-votre-ajout

# 3. Faire vos modifications et commiter
git commit -m "feat: description courte de la modification"

# 4. Pousser et ouvrir une Pull Request
git push origin feature/nom-de-votre-ajout
```

## Convention de commits

- `feat:` nouvelle fonctionnalité
- `fix:` correction de bug
- `docs:` documentation
- `config:` fichiers de configuration

## Code de conduite

Soyez respectueux, bienveillant et constructif. Ce projet est ouvert à tous les niveaux.
